Hola Mundo in Go

Here is your first go program.

package main

import "fmt"

func main() {

	phrase := "Hola Mundo!"

	fmt.Println(phrase)
}

All programs must be part of a package and main is the default package

Import standard library packages, they are not included by default. The standard library is your best friend in Go, it can do just about everything. See all the glory -> http://golang.org/pkg/

fmt library provides basic standard out formatting functions

the main package requires a main() function which gets called when run

define a variable, all variables have types, but the go compiler will automatically detect the type if possible, this is a string
